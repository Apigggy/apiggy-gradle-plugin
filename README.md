# apiggs-gradle-plugin
easy use apiggs with gradle

## Usage
install build.gradle
```groovy
buildscript {
    dependencies {
        classpath 'com.github.apiggs:apiggs-gradle-plugin:1.0'
    }
}

apply plugin: 'com.github.apiggs'
```

cmd
```groovy
gradle apiggs
```

tasks
```
Tasks/documentation/apiggs
```
